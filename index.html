Swipe Layer tool by @kamicut

Use the URL bar to input two mapbox IDs, a centerpoint and a zoom level in the form

?param1=value1&param2=value2

Parameters

  • left: the left mapbox ID
  • right: the right mapbox ID
  • zoom: the zoom level
  • centroid: latitude, longitude
  • zoomControl: default is false, enable/disable zooming
  • enableInfo: default is false, enable/disable this popup
